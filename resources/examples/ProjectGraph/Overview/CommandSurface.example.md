Project Graph exposes distinct command surfaces once the package is installed:

- `ai:review-graph:show` renders a readable slice of the stored graph.
- `ai:review-graph:query` answers focused dependency and usage questions.
- `ai:review-graph:module` packages a whole-module overview with optional flows and events.
- `ai:review-graph:intelligence` returns higher-level answers such as hotspots or documentation gaps.
- `ai:review-graph:context` builds a task-scoped context package for review, refactor, or AI work.
- `ai:review-graph:impact` estimates downstream blast radius before edits.
